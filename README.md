About the Project

The G3 Architect Website is a practice project to showcase a fictional architecture firm. It includes:

A clean and modern design

A navigation bar for main pages: Home, About, Contact, and Login

Banner section with company description and image

Team section displaying team members

Features section highlighting platform capabilities

Facts and statistics about the company

Sponsors section to show company partners

The goal of this project was to practice structuring a website, responsive layout, and visual presentation.
